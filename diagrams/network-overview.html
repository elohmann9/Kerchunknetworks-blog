Adding Your draw.io Diagrams

To add your network diagrams:

  1. Export diagrams from draw.io as PNG (300 DPI recommended)
  2. Upload to the /diagrams/ folder in your GitHub repository
  3. Update the image sources below to match your filenames
  4. Commit changes to make them live
